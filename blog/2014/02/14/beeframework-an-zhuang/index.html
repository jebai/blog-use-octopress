BeeFramework 安装

Feb 14th, 2014 | Comments

BeeFramework 升级到0.5.0后没有提供xcode模板，手动安装到工程比较麻烦（我没有成功过），推荐使用Pod方式来安装。

从新建一个工程，到成功引用BeeFramework，大概需要以下几个步骤；

  • 新建空白的 iOS Empty Applicaton 工程

  • 在工程中创建Podfile，并配置BeeFramework

  • 执行pod install

  • 创建软链接

    pod安装后生成的工程中，自动添加libPods路径为主工程的build目录中，

    但Pods工程的编辑输出在另一个目录

    可以通过添加软链接来解决问题

  • 添加Bee提供的bundle到主工程中

    这样开发态的一些网格，线框图图标就可以展示出来了

  • Done

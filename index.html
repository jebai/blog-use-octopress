iOS 修改编译输出目录

Dec 21st, 2013

XCode5中默认的工程编译输出目录是 ~/Library/Developer/Xcode/DerivedData。

可以根据需要改变设置，使像静态库这样的编译输出文件可以方便的被找到。

1、在Xcode菜单中按如下路径打开对话框：

XCode->Preferences->Locations

2、点击DerivedData的高级设置选项

3、将BuildLocation设置为Legacy

用octopress构建个人博客

Nov 14th, 2013

1. 下载Octopress

在准备存放仓库的目录将octopress代码下载下来<br/>
`git clone git://github.com/imathis/octopress.git octopress`

2. 安装依赖的组件

  • cd octopress
  • gem install bundler. 了解bundler
  • rbenv rehash 了解rbenv
  • bundle install

3. 安装默认的样式

  • rake install了解rake

4. 在github上新建仓库

仓库需要命名为 username.github.io.
username要跟github用户名相同

5. 在本地新建站点，同时部署站点的github信息

`rake setup_github_pages`

6. 生成站点

`rake generate`

7. 创建一篇文章

`rake new_post["first blog"]
这样会在source/_post目录下面创建一编文章，时间开头，md结尾，Mac下可以用mou来编辑md文件。`

了解mou

8. 本地预览站点

`rake preview`
而后即可在浏览器中访问 http://0.0.0.0:4000

9. 部署站点

`rake deploy`

10. 访问站点

http://username.github.io
